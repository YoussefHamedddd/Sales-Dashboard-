## Project Overview

This repository showcases a comprehensive **Sales Performance Dashboard** created using **Tableau Public**. The project aims to provide an interactive and visual representation of key sales metrics, allowing for quick insights into performance trends, profitability, and geographic distribution.

The dashboard utilizes sample Superstore data (or specify your data source) to analyze:
* Total Sales, Profit, and Quantity.
* Sales performance over time (Monthly and Quarterly trends).
* Geographic sales distribution (Sales by State/Region).
* Product-level performance (Sales and Profit by Sub-Category).
